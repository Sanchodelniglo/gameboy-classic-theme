# Change Log

All notable changes to the "gameboy-classic-theme" extension will be documented in this file.

Check [Keep a Changelog](http://keepachangelog.com/) for recommendations on how to structure this file.

## [0.0.3] - 2026-02-04

### Added
- Comprehensive token colors for all major languages (JS/TS, Python, Ruby, Go, HTML, CSS, JSON, Markdown)
- Semantic highlighting support with `semanticHighlighting: true`
- Git decoration colors (added, modified, deleted, untracked, ignored, conflicting)
- Terminal ANSI colors (all 16 colors)
- Diff editor colors
- Terminal panel colors (sticky scroll, command decorations, tab borders)
- Command palette/quick input colors
- Toolbar hover/active states
- Panel section headers and borders
- 6 bracket pair colors (was 3)
- Scrollbar, minimap, peek view colors

### Fixed
- Comment contrast for WCAG AA compliance (light: #526525, dark: #6a8a3a)
- Dark theme now actually dark (UI backgrounds changed from beige to dark grays #272929, #1e2020)
- Icon foreground contrast on light theme
- Keybinding label contrast
- JSON formatting (removed trailing commas, consistent indentation)

### Changed
- Badge colors adjusted for better visibility
- Panel title foreground improved

## [0.0.2] - Initial Release

- Initial release with light and dark GameBoy Classic themes
