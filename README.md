## Features

- **Two variants**: Light (classic GameBoy screen) and Dark (backlit GameBoy feel)
- **Authentic GameBoy color palette** with the iconic green tones
- **WCAG AA compliant** contrast ratios for comfortable reading
- **Comprehensive syntax highlighting** for all major languages
- **Semantic highlighting support** for enhanced code understanding
- **Full workbench theming** including terminal, git decorations, diff editor, brackets, and more

## Installation

### Via VS Code Marketplace

1. Open the **Extensions** sidebar in VS Code (`Cmd+Shift+X` / `Ctrl+Shift+X`)
2. Search for **"GameBoy Classic Theme"**
3. Click **Install**

### Via Command Palette

1. Open Command Palette (`Cmd+Shift+P` / `Ctrl+Shift+P`)
2. Run `ext install sanchodelniglo.gameboy-classic-theme`

## Activation

1. Open Command Palette (`Cmd+Shift+P` / `Ctrl+Shift+P`)
2. Search for **"Preferences: Color Theme"**
3. Select **"GameBoy Classic - Light"** or **"GameBoy Classic - Dark"**
